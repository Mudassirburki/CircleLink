# 🚀 CircleLink

CircleLink is a modern social networking application designed to connect people seamlessly. Users can create profiles, share updates, and engage with communities in a dynamic environment.

---

## 📱 Features

- 🔐 User Authentication (Login / Signup)
- 📲 Social Login Integration
- 👤 User Profiles
- 🏠 Home Feed
- 🔍 Search & Discover Users

---

## 🛠️ Tech Stack

- **Mobile App:** React Native  
- **Backend:** Firebase / Node.js  
- **Database:** Firestore / MongoDB
